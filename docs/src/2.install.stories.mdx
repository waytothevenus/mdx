# Install

```sh
npm install mdx-embed --save
```

### Install peer dependencies

```js
npm install  mdx-embed @mdx-js/mdx @mdx-js/react --save
```

Add the `MDXEmbedProvider` to your application by importing it from `mdx-embed` and wrapping your applications `MDXRenderer`.

```javascript
import { MDXEmbedProvider } from 'mdx-embed';

<MDXEmbedProvider>
  <MDXRenderer>{body}</MDXRenderer>
</MDXEmbedProvider>;
```
