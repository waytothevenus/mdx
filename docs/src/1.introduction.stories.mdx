# MDX Embed

MDX Embed allows you to easily _embed_ popular 3rd party media content such as YouTube videos, Tweets, Instagram posts and many more straight into your `.mdx` - **no import required!**

<Divider />

<Link href="https://github.com/PaulieScanlon/mdx-embed" target="_blank">

</Link>

## Usage Example

```js
// some-mdx-file.mdx
#### My cool pen

Here's a pen, and some other blog post text

<CodePen codePenId="PNaGbb" />
```
