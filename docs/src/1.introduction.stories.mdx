# MDX Embed

MDX Embed allows you to easily _embed_ popular 3rd part media content such as YouTube videos, Tweets, Instagram posts and many more straight into your `.mdx` - **no imports required!**

<Divider />

<Link href="https://github.com/PaulieScanlon/mdx-embed" target="_blank">
  <Button>GitHub</Button>
</Link>

## Usage Example

```js
// some-mdx-file.mdx
#### My cool pen

Here's a pen, and some other blog post text

<CodePen codePenId="PNaGbb" />
```
